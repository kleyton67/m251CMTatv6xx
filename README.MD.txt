# Programação de CLP M251MESE
Neste repositório está alguns exemplos de comunicação com Inversores ATV6xx, contendo exemplo de bloco de comunicação, Leitura e Escrita

## Configuração
Utilizamos Perfil I/O 2-Wire para comunicação com os equipamentos;
Utilizamos Modbus Ethenet para comunicação com os equipamento em rede;
Utilizamos Ethernet no registradores CD1 e CD2;
Utilziamos Referência ao CD1 no registrador RCB;

## Fontes
utilizamos como exemplo para configuração um arquivo de conexão mais simples encontrado no arquivo example.st;
